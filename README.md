# tg-auto-msg
# Telegram Auto-Messenger v1.0
**Developed by c4hex**

Bu yazılım Telegram kanallarına ve gruplarına otomatik mesaj göndermenizi sağlar. 
Tamamen terminal tabanlıdır, hafiftir ve her türlü Linux/Windows/VPS ortamında çalışır.

### 🛠️ Kurulum
1. Önce kütüphaneyi kurun:
   `pip install telethon`
2. [my.telegram.org](https://my.telegram.org) adresinden `api_id` ve `api_hash` alın.
3. `main.py` dosyasındaki ilgili alanları doldurun.

###  Kullanım
`python3 main.py`

* İlk girişte telefon numarası (+90...) ve SMS kodu isteyecektir.
* Bir kez giriş yaptıktan sonra `session` dosyası oluşturur, bir daha sormaz.
